Prototypes to plot timelines with gnuplot of RIPE Atlas and RIS data for specific geographies.

doit-probe-connects.py
  plot probe up/down to plot over time. Needs gnuplot installed locally + python libraries at top of script.

doit-routing-sg-inrdb.py
  uses sg-inrdb to plot ASNs. won't work if you don't have access to SG-INRDB!

cluster-csv.py
  prototype clustering algo, similar to John Heidemann in Trinocular paper
